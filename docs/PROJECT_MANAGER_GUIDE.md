# PROJECT MANAGER GUIDE

## Dashboard Overview

### Executive Dashboard
- **Revenue**: Total invoiced, collected, outstanding
- **Billing**: Invoices generated, paid, overdue
- **Customers**: Active, inactive, new, delinquent
- **Meters**: Total by type, health %, disconnected
- **Projects**: Performance comparison across all projects

### KPI Dashboards
| Dashboard | Purpose |
|-----------|---------|
| Executive | High-level business performance |
| Collections | Collection rate, aging buckets, recovery |
| Utilities | Consumption by utility type |

## Monthly Billing Cycle

### Step-by-Step

```
Day 1-5:  Collect readings (manual + automated)
Day 6:    Review and approve readings
Day 7:    Create bill cycle (OPEN)
Day 8:    Lock bill cycle → generate invoices
Day 9:    Review generated invoices
Day 10:   Approve and post invoices
Day 11-25: Process payments as they arrive
Day 26:   Run month-end reports
Day 27:   Close bill cycle
```

## Reports Available

| Category | Reports |
|----------|---------|
| Financial | Invoices Summary, Payments, Aging, Collection Efficiency |
| Operations | Meters Status, Reading Register, High/Low Consumption |
| Customer | Customer List, Customer Aging, Customer History |
| Utilities | Water Balance, Solar Generation, Meter Health |
| System | Audit Log, User Activity, System Config |

## Project Performance

Monitor these KPIs for each project:
- Collection rate (target: >85%)
- Outstanding balance (target: decreasing)
- Active meter count
- New customer signups per month
