# CUSTOMER SERVICE GUIDE

## Common Tasks

### 1. Find Customer
- Search by: name, code, phone, national ID, meter serial
- Go to **Customers** → enter search term
- Results show matching customers with account status

### 2. View Customer Details
Customer detail page has these tabs:

| Tab | What You See |
|-----|--------------|
| **Overview** | Name, code, status, balance, contact info |
| **Meters** | All meters assigned to customer |
| **Invoices** | All invoices with status and amounts |
| **Payments** | Payment history with methods |
| **Ledger** | Running balance with all transactions |
| **Wallet** | Prepaid/credit wallet balance |
| **Ownership** | Ownership transfer history |

### 3. Handle Complaints
1. Go to **Tickets** → **New Ticket**
2. Select customer
3. Choose category: billing, meter, payment, other
4. Enter description
5. Set priority: low/medium/high/critical
6. Submit → ticket created with unique ID

### 4. Transfer Ownership
1. Go to customer detail → **Ownership** tab
2. Click **Transfer Ownership**
3. Search for new owner customer
4. Enter reason for transfer
5. Confirm → system transfers all meters, invoices, wallet
6. Source customer deactivated automatically

### 5. Update Customer Info
1. Go to customer detail → click **Edit**
2. Update fields: phone, email, address
3. Submit changes

## Customer Statuses

| Status | Meaning | Can Bill? |
|--------|---------|-----------|
| Active | Normal operation | Yes |
| Inactive | Account closed/transferred | No |
| Suspended | Temporary block | No (requires reactivation) |

## Escalation Path

1. Level 1: Customer Service handles initial contact
2. Level 2: Team Leader — payment disputes, complex billing
3. Level 3: Manager — legal issues, permanent account changes
