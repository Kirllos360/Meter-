Summary: What needs to be installed from scratch

If starting from a clean machine, install in this order:
  1. Node.js v24.15.0 — runtime foundation
  2. Docker Desktop — PostgreSQL container
  3. Bun v1.3.14 — frontend runtime
  4. Python 3.12+ (via uv python install 3.12) — for Speckit & Graphify
  5. Speckit: uv tool install specify-cli --from git+https://github.com/github/spec-kit.git@v0.8.13
  6. Graphify: uv tool install graphifyy then graphify install --platform opencode
  7. Frontend: cd Frontend && npm install (or bun install)
  8. Backend: cd backend && npm install
  9. Docker: cd backend && docker compose up -d db (for PostgreSQL)
  10. Playwright: cd Frontend && npx playwright install chromium
